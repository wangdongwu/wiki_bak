ECMAScript 变量可能包含两种不同数据类型的值：基本类型值和引用类型值。

**基本类型值** 指的是那些保存在 **栈内存** 中的简单数据段，即这种值完全保存在内存中的一个位置。

**引用类型值** 指的是那些保存在 **堆内存** 中的对象，意思是变量中保存的实际上只是一个指针，这个指针指向内存中的另一个位置，该位置保存对象。
